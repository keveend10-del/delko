# Delivery SOP

## Month 1

### Week 1 — Foundation
- [ ] Full audit: GBP, website, reviews, analytics, tracking
- [ ] Get all access: Google Business, website CMS, GA4, any existing CRM
- [ ] Set up analytics and conversion tracking
- [ ] Set up review request system
- [ ] Send client kickoff confirmation with what to expect

### Week 2 — Core Fixes
- [ ] GBP optimization: photos, categories, hours, description, Q&A, services
- [ ] Website fixes: mobile speed, call button, contact form, key CTAs, service info
- [ ] Lead form installed and tested
- [ ] Conversion tracking confirmed working

### Week 3 — Content + Follow-Up
- [ ] Content calendar drafted and approved
- [ ] First posts published
- [ ] Review ask templates set up (text / email)
- [ ] Email or SMS follow-up sequences active
- [ ] Missed call text-back set up (if on Local Growth System or above)

### Week 4 — Report + Review
- [ ] Month 1 report sent
- [ ] Performance review call scheduled
- [ ] Next month plan drafted
- [ ] Testimonial ask if early results are strong

---

## Monthly Report Format

Keep it one page. No fluff.

**[Client Name] — [Month Year]**

### What we worked on this month
[3–5 bullet points]

### Calls / leads / forms / messages
[Numbers. Month over month if available.]

### GBP performance
[Views, calls, direction requests. vs. prior month.]

### Website performance
[Sessions, top pages, form submissions.]

### Reviews gained
[Number of new reviews, current total, current rating.]

### Content posted
[Number of posts, top performer if trackable.]

### AI / search visibility notes
[Any changes to how the business appears in AI tools or local search.]

### Blockers
[What slowed progress or is still pending from the client side.]

### Next month's plan
[3–5 specific things we're doing next month.]

---

## Handoff Checklist (End of Month 1)

- [ ] All access documented in CRM / client record
- [ ] GBP fully optimized and claimed
- [ ] Website fixes live and tested on mobile
- [ ] Review system sending requests
- [ ] Follow-up sequences active
- [ ] Tracking confirmed
- [ ] Month 1 report sent
- [ ] Next month's plan shared
